# Storm Response Dashboard

A comprehensive dashboard that combines Samsara fleet tracking with SiteFotos forms and photos for emergency storm response coordination.

## Features

- 🚛 **Real-time Fleet Tracking** (Samsara API)
- 📝 **Live Form Submissions** (SiteFotos API)  
- 📸 **Storm Damage Photos** (SiteFotos API)
- 🖥️ **Smart Display Control** (SmartThings API)
- 🎮 **ViKi Stream Dock Integration**
- 📡 **Real-time WebSocket Updates**

## Quick Start

### 1. Installation

```bash
# Extract the deployment package
